1. Write a blog on Difference between HTTP1.1 vs HTTP2 ?
           
      # HTTP/1.1 and HTTP/2 are versions of the Hypertext Transfer Protocol (HTTP), which is used for communication on the World Wide Web.

      # On difference between as

             * HTTP/1.1
         + It’s a text-based protocol where requests and responses are sent in plain text.
         + It allows only one request to be handled at a time per TCP connection.

             * HTTP/2
         + It’s a binary protocol, which makes it more efficient for computers to parse.
         + It introduces multiplexing, allowing multiple requests and responses to be sent over a single TCP connection.

      # Yes I read about IP address, Port, HTTP Methods & MAC Address.
      # To Know,

		+ IP ADDRESS - Network address for the computer.
		+ PORT - a virtual point where network communication happens between two devices to send & receive data.
		+ HTTP Methods - GET, PUT, POST, DELETE, PATCH.
		+ MAC Address - Unique identifier assigned to a network interface controller (NIC) for use as a network address in communications within a network segment.
